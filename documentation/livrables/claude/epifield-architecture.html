🎯 Résumé Exécutif

EpiField est une plateforme de surveillance épidémiologique multi-pathogènes construite sur une architecture pipeline linéaire en 3 couches : Ingestion → Traitement → Présentation. Le projet cible actuellement le MPXV (Monkeypox) avec une architecture généraliste extensible à tout pathogène. La plateforme est déployée gratuitement sur Streamlit Community Cloud + GitHub, sans infrastructure serveur.

✨ Architecture en 30 secondes : CRF Word / rdmStats.py → SWITCH MODE → EXTRACTION → TRAITEMENT → DASHBOARD STREAMLIT → Utilisateurs

🔀 Switch Synthétique ↔ Réel
Mécanisme clé · 1 variable d'environnement · Sécurité by design
🔑 Concept Central : Le switch DATA_MODE est la pièce maîtresse de l'architecture. Il permet de passer instantanément entre données fictives (développement) et données réelles CRF (production), sans modifier le code.
🔄
MODE SYNTHÉTIQUE
DATA_MODE = "synthetic"
Défaut
✅ 900 cas fictifs réalistes
✅ Zéro risque RGPD
✅ Tests sans données sensibles
✅ Idéal pour développement & démo
✅ Saisonnalité réaliste incluse
export DATA_MODE=synthetic
# → rdmStats.generate_synthetic_data()
⇄
🚨
MODE RÉEL
DATA_MODE = "real"
Production
✅ Vraies données CRF patients
✅ Après validation RGPD
✅ Environnement production
✅ Secret GitHub (sécurisé)
✅ Données anonymisées versionnées
export DATA_MODE=real
# → pd.read_json("extraction_real.json")
💻 Implémentation — Code Complet
dashboard_app.py · ligne ~50
# =================================================== # SWITCH SYNTHÉTIQUE ↔ RÉEL — Cœur de l'architecture # =================================================== import os import pandas as pd import rdmStats # 1 ligne de code pour basculer entre les modes MODE = os.getenv("DATA_MODE", "synthetic") ← Variable env if MODE == "synthetic": # Dev/Test : 900 cas fictifs (rapide, sûr, zéro RGPD) data = rdmStats.generate_synthetic_data() print("🔄 Mode SYNTHÉTIQUE activé (développement)") else: # Production : vraies données CRF après nettoyage RGPD data = pd.read_json("extraction_real.json") print("🚨 Mode RÉEL activé (production)") # Le reste du pipeline est identique dans les deux cas ! # Normalisation → Analyse → Dashboard → Streamlit

📖 Glossaire
CRF — Case Report Form. Formulaire médical de collecte des données patients.
MPXV — Monkeypox Virus. Pathogène cible du prototype EpiField.
DATA_MODE — Variable d'environnement contrôlant le switch synthétique ↔ réel.
Streamlit — Framework Python pour créer des dashboards web (déploiement cloud gratuit).
Prophet — Modèle de prévision temporelle (Facebook/Meta). Détecte tendances et saisonnalité.
Serverless — Déploiement sans gestion de serveur. Auto-scaling, maintenance réduite.
